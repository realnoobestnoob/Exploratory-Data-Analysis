# Singapore Public Hospital Admissions Analysis

## Overview

This project analyses monthly admissions to Singapore public hospitals using healthcare datasets from data.gov.sg.

The analysis explores:
- Long-term admission trends
- Seasonal healthcare demand
- Post-pandemic healthcare recovery
- Potential implications for healthcare planning

---

# Objectives

Key questions explored:
- How have hospital admissions evolved over time?
- Are there seasonal demand patterns?
- How did COVID-19 impact admissions?
- What trends may affect Singapore’s healthcare system in the future?

---

# Dataset

Source:
- data.gov.sg

---

# Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Key Insights

- Long-Term Healthcare Demand is Increasing
- COVID-19 Created Significant Short-Term Disruptions
- Seasonal Patterns Exist in Admissions
- Admission Patterns Among Certain Hospitals Are Highly Correlated
---

# Conclusion

This project highlights how healthcare data can be used to better understand public healthcare demand and support long-term healthcare planning in Singapore.
